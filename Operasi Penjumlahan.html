Operasi Bilangan Bulat

Penjumlahan dan Pengurangan Bilangan Positif, Nol, dan Negatif

Pendahuluan

Bilangan bulat adalah himpunan bilangan yang terdiri dari bilangan positif, nol, dan bilangan negatif. Simbolnya adalah Z = {..., -3, -2, -1, 0, 1, 2, 3, ...}

Contoh:

- Bilangan positif: 1, 2, 3, 4, ...

- Bilangan negatif: -1, -2, -3, -4, ...

- Bilangan nol: 0

Penjumlahan Bilangan Bulat

Aturan Penjumlahan:

1. Positif + Positif: Hasilnya positif (contoh: 3 + 4 = 7)

2. Negatif + Negatif: Hasilnya negatif (contoh: -3 + (-4) = -7)

3. Positif + Negatif: Kurangi nilai absolut yang lebih besar dengan yang lebih kecil, tanda hasil sesuai bilangan yang memiliki nilai mutlak lebih besar (contoh: 5 + (-3) = 2 karena |5| > |-3|)

4. Negatif + Positif: Sama seperti aturan 3 (contoh: -5 + 3 = -2 karena |-5| > |3|)

Pengurangan Bilangan Bulat

Aturan Pengurangan:

Pengurangan bilangan bulat dapat diubah menjadi penjumlahan dengan cara: a - b = a + (-b)

Contoh:

  • 5 - 3 = 5 + (-3) = 2
  • 5 - (-3) = 5 + 3 = 8
  • -5 - 3 = -5 + (-3) = -8
  • -5 - (-3) = -5 + 3 = -2

Ringkasan Materi

  • Penjumlahan: Untuk menjumlahkan dua bilangan bulat, kita bisa menggabungkan nilainya dengan mempertimbangkan tanda.
  • Pengurangan: Pengurangan bilangan bulat sama dengan penjumlahan dengan lawannya.
  • Visualisasi: Gunakan garis bilangan untuk membantu memahami operasi bilangan bulat.
  • Prinsip Dasar: Operasi bilangan bulat mengikuti aturan tertentu yang harus dipahami dengan baik.
